Getting Dates

At the end of each game section -- that is, after each set of eight questions -- is a landing where you can stop and see the dates you've found arranged chronologically. If you'd like to see any of the timelines first, you can go directly to each landing from this page. Each subsequent timeline includes the dates from the earlier ones, so if you want to see all the dates at once, you can go directly to TIMELINE 4.
